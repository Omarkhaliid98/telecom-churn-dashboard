# 📊 Telecom Churn & Customer Behavior Dashboard

This project is a **Power BI dashboard** that gives telecom companies the full picture about **why customers leave (churn)** and how to **keep them longer**.

## 🔍 What You’ll Find Inside:
- Retention & churn KPIs
- Lost revenue analysis by offer
- Churn reasons (price, service, support...)
- Demographic analysis (age, gender, marital status)
- Contract types (monthly, annual, 2-year)
- Tenure-based churn trends
